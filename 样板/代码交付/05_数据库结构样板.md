## 4. 表结构定义

### 4.1 表名：video_task

表说明：视频任务表。

字段清单：

| 字段名 | 类型 | 是否非空 | 默认值 | Java 字段 | API 字段 | 说明 |
|---|---|---|---|---|---|---|
| id | BIGINT | 是 | 无 | id | id | 主键 |
| title | VARCHAR(200) | 是 | 无 | title | title | 任务标题 |
| topic | VARCHAR(200) | 是 | 无 | topic | topic | 视频主题 |
| script_summary | VARCHAR(1000) | 否 | NULL | scriptSummary | scriptSummary | 脚本摘要 |
| storyboard_summary | VARCHAR(1000) | 否 | NULL | storyboardSummary | storyboardSummary | 分镜摘要 |
| status | VARCHAR(50) | 是 | draft | status | status | 任务状态 |
| created_at | TIMESTAMP | 是 | 无 | createdAt | createdAt | 创建时间 |
| updated_at | TIMESTAMP | 是 | 无 | updatedAt | updatedAt | 更新时间 |

主键：

```text
id
```

索引：

| 索引名 | 字段 | 类型 | 查询场景 |
|---|---|---|---|
| idx_video_task_status | status | 普通索引 | 按状态筛选任务 |
| idx_video_task_created_at | created_at | 普通索引 | 按创建时间排序 |

唯一约束：

```text
无
```

状态枚举：

| 值 | 说明 |
|---|---|
| draft | 草稿 |
| pending | 等待处理 |
| generating | 生成中 |
| completed | 已完成 |
| failed | 失败 |

是否软删除：否

是否审计：否

对应 Entity：

```text
VideoTask
```

对应 Mapper：

```text
VideoTaskMapper
```

对应接口：

```text
GET /api/v1/video-tasks
GET /api/v1/video-tasks/{id}
POST /api/v1/video-tasks
PUT /api/v1/video-tasks/{id}
```

## 5. Flyway migration

对应 migration 文件：

```text
src/main/resources/db/migration/V1__create_video_task.sql
```

规则：

- 新增表必须新增 migration。
- 修改表结构必须新增 migration。
- 不得修改已经执行过的 migration。

## 8. AI 生成约束

AI 生成或修改数据库结构时，必须：

- 使用本文件定义的表名和字段名。
- 同步更新接口契约。
- 同步更新 Entity、BO、VO、Mapper 和 Service。
- 新增 Flyway migration。
- 不直接把 Entity 返回给前端。
