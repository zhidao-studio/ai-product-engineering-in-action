# 交付迭代约束样板

## 1. 交付条件

- 产品定义已确认
- 设计规格已确认
- 工程规格已确认
- 工程基线已确认
- 验证已通过
- 体验验收已通过

## 2. 上线前检查

- 页面或服务可打开
- 核心流程可完成
- 空状态和错误状态可显示
- 没有引入不做清单里的能力

## 3. 反馈分类

| 反馈类型 | 处理方式 |
|---|---|
| 缺陷 | 进入修正任务 |
| 体验问题 | 更新体验验收或设计规格 |
| 范围变化 | 先更新产品定义 |
| 新需求 | 放入下一版本评估 |
