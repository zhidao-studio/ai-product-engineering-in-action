# 版本路线图

## 当前定位

本仓库是《AI 产品工程实战》的课程、模板、提示词和实战样例仓库。

当前主线目标：

```text
先把一个微信小程序实战样例讲清楚、跑通、可跟做，再扩展更多产品类型。
```

## v0.1：初始化骨架

已完成：

- 仓库目录结构
- 课程总说明
- 方法论初稿
- 文档模板
- Prompt 初稿
- 微信小程序样例初稿

## v0.2：可学习课程骨架

已完成：

- 补齐 6 节 MVP 课程正文
- 加入产品验证
- 加入非技术用户决策树
- 增强提示词
- 增加问题排查
- 增加模拟用户验收机制
- 从强绑定 Codex 改成 AI 编码工具可替换

## v0.3：图文课程版本

已完成：

- 补齐 6 节图文课程
- 增加流程图、页面示意、输入示例、输出示例
- 补充错误案例和常见错误
- 明确图文课程为主学习入口
- 收敛课程结构，去掉多余学习层级

## v0.3.3：发布前结构收尾

已完成：

- README 更新到当前版本
- `docs/02_lessons/visual/` 明确为唯一主课程入口
- `lesson-xx` 基础正文定位为原理补充
- 删除多余 `hands-on-lesson-*` 层级
- 避免 `Demo / 演示版本` 等临时性表述

## v0.4：真实截图与标注版

下一步重点：

- 替换图文课程中的截图占位
- 补充微信开发者工具真实截图

- 为截图增加标注：看哪里、点哪里、复制哪里、检查哪里
- 完成首页 → 详情页 → 收藏 → 收藏页的完整闭环截图
- 让课程达到早期用户试学标准

## v0.5：多样例扩展

计划新增：

- H5 落地页样例
- 简单 Web App 样例
- 浏览器插件样例
- iOS App 进阶样例

## v1.0：系统课程版本

目标：

- 课程可完整学习
- 模板可直接复用
- 样例工程可运行
- 有完整排错手册
- 有完整验收机制
- 有真实截图和标注
- 有至少 2-3 个可参考实战样例
