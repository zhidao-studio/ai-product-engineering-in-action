# 版本路线图

## 当前定位

本仓库是《AI 产品工程实战》的课程、模板、提示词和实战样例仓库。

当前主线目标：

```text
构造一套 AI 驱动交付 Harness，
用产品、设计、工程、任务、验证、体验和迭代等多层约束，
控制 AI 如何交付产品。
```

样例形态可以变化，Harness 架构不变。
