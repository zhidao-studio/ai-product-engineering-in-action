# 版本路线图

## 当前定位

本仓库是《AI 产品工程实战》的课程、模板、提示词和实战样例仓库。

当前主线目标：

```text
用一整套完整文档控制需求如何被实现，
再让 Agent 在文档边界内执行任务。
```

样例形态可以变化，文档控制体系不变。

- 仓库目录结构
- 课程总说明

- 文档模板
- Prompt 初稿
- 第一版样例初稿
