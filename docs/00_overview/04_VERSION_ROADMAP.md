# 版本路线图

## 当前定位

本仓库是《AI 产品工程实战》的课程、模板、提示词和实战案例仓库。

## v0.1：初始化骨架

已完成：

- 仓库目录结构
- 课程总说明
- 方法论初稿
- 文档模板
- Prompt 初稿
- 微信小程序案例初稿

## v0.2：可学习课程骨架

目标：

- 补齐 6 节 MVP 课程正文
- 加入产品验证
- 加入非技术用户决策树
- 增强提示词
- 增加问题排查
- 增加模拟用户验收机制
- 从强绑定 Codex 改成 AI 编码工具可替换

## v0.3：第一个可公开学习版本

计划：

- 补充每节课的图文演示
- 完善小程序示例代码讲解
- 补充错误案例
- 补充练习题
- 增加学员作业模板

## v0.4：多案例扩展

计划新增：

- H5 落地页案例
- 简单 Web App 案例
- 浏览器插件案例
- iOS App 进阶案例

## v1.0：系统课程版本

目标：

- 课程可完整学习
- 模板可直接复用
- 示例项目可运行
- 有完整排错手册
- 有完整验收机制
