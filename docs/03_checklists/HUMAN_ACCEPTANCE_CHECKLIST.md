# 模拟用户验收检查清单

## 1. 用户理解

- [ ] 第一次打开能看懂这是做什么的。
- [ ] 页面标题清楚。
- [ ] 按钮文案清楚。
- [ ] 空状态有引导。
- [ ] 错误状态有解释。

## 2. 核心流程

- [ ] 用户能从首页开始操作。
- [ ] 用户能进入详情页。
- [ ] 用户能完成核心动作。
- [ ] 用户能看到动作结果。
- [ ] 用户能返回或继续下一步。

## 3. 逻辑一致性

- [ ] 收藏后状态正确。
- [ ] 取消收藏后状态正确。
- [ ] 收藏页数据正确。
- [ ] 重新打开后数据仍符合预期。
- [ ] 没有页面死路。

## 4. 体验判断

- [ ] 1 分钟内能完成核心流程。
- [ ] 不需要阅读说明书才能使用。
- [ ] 没有明显让人困惑的地方。
- [ ] 不会让用户误解产品能力。

## 5. 结论

- [ ] 可以交给真实用户试用。
- [ ] 需要先修复关键问题。
- [ ] 暂不适合交付。
