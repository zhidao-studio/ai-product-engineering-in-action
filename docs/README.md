# 文档入口

## 1. 本仓库解决什么问题

本仓库不是一个具体产品样例，也不是某个 AI 编码工具教程。

它要定义一套：

```text
文档驱动的 AI 产品交付方法论。
```

目标是提供可复用的 harness 模板和 checklist，让非技术人能通过文档工程约束 AI，完成可验收、可追溯、可维护的产品交付。

## 2. 最重要的一句话

```text
不是让 AI 替你做产品，
而是用 harness 约束 AI 完成可验收的产品交付。
```

## 3. 三个目标层级

| 优先级 | 目标 | 说明 |
|---:|---|---|
| 1 | 方法论目标 | 定义文档驱动的 AI 产品交付流程 |
| 2 | 工具化目标 | 提供可复用的 harness 模板和 checklist |
| 3 | 教学目标 | 让非技术人能学会并实际使用 |
