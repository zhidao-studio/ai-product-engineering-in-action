# 文档导航

本目录只放说明性文档。

模板、检查清单、提示词、示例分别放在仓库顶层对应目录，不放进 docs。

## 1. 目录职责

```text
docs/methodology/  方法论：目标、原则、分层、构造方法、覆盖检查
docs/engineering/  工程基线：技术选择、分层规则、质量红线
docs/lessons/      教学化内容：后续面向非技术人的课程化表达
```
