# 图文课程

这里是当前课程的主学习入口。

## 当前学习建议

当前学习请以 `visual/` 目录下的图文版为主。

原 `lesson-xx` 文件只保留用于理解概念和原则。

## 图文课目标

让非技术学员按图文步骤理解：

```text
如何通过一整套完整文档，控制需求如何被实现。
```

课程重点不是某个具体样例工程，也不是某个固定 Agent，而是文档控制链路。

## 关键修正

文档控制不是单线流程。

页面范围、功能范围、交互流程、数据字段、设计原型、前端实现、后端实现、接口规范需要一起考虑。

代码工程执行层也不是一个“Agent 实现”就够了，而要拆成：

```text
设计 / 原型
前端
后端
接口 / 联调
审查 / 验收
```

## 图文课结构

每一课固定包含：

1. 本节目标
2. 本节产物
3. 流程图
4. 操作步骤
5. 输入示例
6. 输出示例
7. 截图位置
8. 检查清单
9. 常见错误
10. 下一步

## 截图说明

真实截图放在 v0.4.x 补充。

当前版本先用：

```text
[截图占位：这里放 xxx 截图]
```

后续截图重点不是展示某个平台工具，而是展示：

- 文档如何生成
- 页面、流程、数据和原型如何互相校验
- 前端 / 后端 / 接口任务如何拆分
- Agent 如何按任务类型执行
- 实现如何回溯到文档
- 用户验收如何发现问题并转成任务
