# 图文课程

这里是 v0.3 图文课程区，也是当前课程的主学习入口。

## 当前学习建议

当前学习请以 `visual/` 目录下的图文版为主。

不要再切换到 `hands-on-lesson-*` 或旧的 `demo-lesson-*` 文件。

原 `lesson-xx` 文件只保留用于理解概念和原则。

## 图文课目标

让非技术学员不用先理解大量概念，也能按图文步骤完成第一版实战样例。

## 图文课结构

每一课固定包含：

1. 本节目标
2. 本节产物
3. 流程图
4. 操作步骤
5. 输入示例
6. 输出示例
7. 截图位置
8. 检查清单
9. 常见错误
10. 下一步
