# 教学化内容

本目录用于后续教学化表达。

当前优先级不是继续维护旧课程，而是在方法论、模板和 checklist 稳定后，再把它改写成非技术人能学会的课程。

## 教学目标

```text
让非技术人学会使用文档驱动的 AI 产品交付 harness。
```

## 后续课程结构建议

| 课时 | 主题 | 对应目标 |
|---|---|---|
| 第 1 课 | 为什么不能直接让 AI 做产品 | 理解 harness 的必要性 |
| 第 2 课 | 如何定义产品边界 | 学会产品定义 Harness |
| 第 3 课 | 如何约束设计和原型 | 学会设计规格 Harness |
| 第 4 课 | 如何约束工程实现 | 学会工程规格 Harness |
| 第 5 课 | 如何让 AI 只做一个任务 | 学会任务执行 Harness |
| 第 6 课 | 如何验收 AI 产物 | 学会验证和体验验收 Harness |
| 第 7 课 | 如何管理上线后的变化 | 学会交付迭代 Harness |
