# 图文课程截图资产

这里存放 v0.4 图文课程需要使用的截图和标注图。

## 当前策略

v0.4 分两步推进：

1. 先放入可替换的 SVG 标注图，让课程先具备图文阅读体验。
2. 后续用微信开发者工具、AI 对话窗口、代码编辑器的真实截图替换同名图片。

## 目录规划

```text
assets/
├── lesson-01/
├── lesson-02/
├── lesson-03/
├── lesson-04/
├── lesson-05/
└── lesson-06/
```

## 命名规则

```text
step-01-xxx.svg
step-02-xxx.png
step-03-xxx.png
```

说明：

- `svg`：当前可替换标注图。
- `png`：后续真实截图。
- 替换真实截图时，建议保持语义命名，不要使用 `image1.png` 这类无意义名称。

## 标注原则

每张图至少标注：

- 看哪里
- 点哪里
- 复制哪里
- 检查哪里
- 完成后进入哪一步
