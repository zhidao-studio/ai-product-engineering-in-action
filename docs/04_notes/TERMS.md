## Agent 和 AI 编码工具的关系

`Agent` 是更宽泛的概念，表示“能按任务执行的 AI 助手”。

本课程为了降低非技术学员理解成本，正文中优先使用更直白的说法：

```text
AI 编码工具
```

它特指能进入工程、读取文件、修改代码、修复 bug 的工具，例如：

- Codex
- Cursor
- Windsurf
- Kimi Coding
- Trae / MarsCode
- GitHub Copilot

因此，学员可以这样理解：

```text
Agent 是大概念；
AI 编码工具是本课程中最常用的一类 Agent。
```

## 课程统一表述

本课程统一使用：

- 样例
- 实战样例
- 样例工程
- 第一版可运行样例
- 第一版可运行样例产品

避免使用：

- Demo
- App Demo
- 小程序 Demo
- 演示版本
- 演示样品

原因：

课程不是展示一个临时效果，而是带学员完成一个结构清晰、可运行、可继续维护的样例产品。
