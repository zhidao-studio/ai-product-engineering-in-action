# 问题排查手册

## 1. AI 生成的代码跑不起来怎么办？

按顺序做：

1. 复制完整报错。
2. 不要让 AI 大范围重写。
3. 用修复提示词让 AI 只修当前错误。
4. 如果连续修 3 次还不对，停止修复，回到任务定义。

修复提示词：

```text
请只修复下面这个报错，不要做无关重构。
请说明根因、修改文件和验证方式。

报错：
【粘贴报错】
```

## 2. 怎么判断是 AI 的问题还是自己的问题？

优先检查：

- 是否打开了正确项目目录。
- 是否复制漏文件。
- 是否修改了错误目录。
- 是否没有保存文件。
- 是否小程序开发者工具缓存未刷新。
- 是否 AI 修改了无关配置。

## 3. AI 越修越乱怎么办？

立刻停止。

处理方式：

1. 查看 git diff。
2. 回滚无关修改。
3. 把任务拆小。
4. 重新给 AI 明确允许修改范围。
5. 必要时回到上一个 commit。

## 4. 什么时候该放弃当前方案？

满足任一情况就要停：

- AI 连续 3 次修复失败。
- 任务已经超出第一版范围。
- 需要引入复杂后端、登录、支付、地图。
- 一个小功能变成多个大改动。
- 你已经看不懂改了什么。

## 5. 微信开发者工具打不开项目

检查打开的是：

```text
examples/01_wechat_mini_program_favorites/miniapp
```

不是整个仓库根目录。

## 6. 页面显示空白怎么办？

检查：

1. `app.json` 是否配置了页面路径。
2. 页面目录是否包含 `.js`、`.wxml`、`.wxss`、`.json`。
3. 控制台是否有报错。
4. 数据是否为空。
5. 是否写错了字段名。
