# 示例：设计规格 Harness

## 1. 视觉风格

简洁、清爽、适合移动端阅读。

## 2. 配色

- 主色：绿色或蓝绿色
- 背景色：浅色
- 文字色：深灰
- 状态色：收藏成功使用明确高亮

## 3. 页面模块

首页：

```text
标题
地点卡片列表
收藏状态标记
```

详情页：

```text
地点名称
地点说明
标签
收藏按钮
```

收藏页：

```text
已收藏地点列表
空状态提示
```

## 4. 页面状态

- 列表为空时显示空状态
- 收藏成功时按钮状态变化
- 收藏列表为空时提示用户先去收藏

## 5. AI 不能自行扩展

AI 不得自行改变页面数量，不得增加复杂视觉风格，不得新增设计规格没有定义的模块。
