# 示例：工程规格 Harness

## 1. 第一版技术边界

第一版只做静态页面或轻量前端。

不做后端。

数据使用 Mock 数据或本地存储。

## 2. 数据字段

```text
id：地点唯一标识
name：地点名称
description：地点说明
tags：地点标签
isFavorite：是否收藏
```

## 3. 前端范围

- 展示地点列表
- 展示地点详情
- 切换收藏状态
- 展示收藏列表
- 展示空状态

## 4. 后端范围

第一版不做后端。

后端能力进入后续版本。

## 5. 接口范围

第一版不做真实 API。

使用 Mock 数据模拟地点列表。

## 6. AI 不能自行扩展

AI 不得自行接入数据库、登录、地图、后端服务或第三方接口。
