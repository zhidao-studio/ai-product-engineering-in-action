# 示例

本目录存放填好的示例。

示例的作用不是展示完整代码工程，而是展示模板应该如何填写。

## 目录说明

| 目录 | 作用 |
|---|---|
| `01_minimal_harness/` | 最小可用 harness 示例 |

## 使用方式

1. 先看 `01_minimal_harness/`。
2. 理解七层 harness 如何被填写。
3. 对照 `templates/harness/` 填写自己的版本。
4. 用 `checklists/harness/` 检查完整性。

## 原则

```text
示例不是为了证明 AI 能写代码，
而是为了证明 harness 能约束 AI 交付。
```
