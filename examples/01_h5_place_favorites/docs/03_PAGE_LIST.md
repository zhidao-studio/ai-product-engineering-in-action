# 页面清单

## 页面总览

| 页面编号 | 页面名称 | 页面入口 | 页面目标 | 对应功能 |
|---|---|---|---|---|
| P001 | 首页 | 默认进入 | 展示地点列表 | F001 |
| P002 | 详情页 | 首页地点卡片 | 展示地点详情和收藏按钮 | F002 / F003 |
| P003 | 收藏页 | 顶部导航 | 展示已收藏地点 | F004 |
| P004 | 设置页 | 顶部导航 | 清空收藏和基础说明 | F005 |
| P005 | 关于页 | 顶部导航 | 展示课程样例说明 | F006 |

## P001：首页

- 页面目标：让用户看到地点列表。
- 页面入口：打开 H5 后默认进入。
- 主要模块：标题区、课程样例说明、地点列表、地点卡片。
- 用户动作：点击地点卡片进入详情页。
- 状态：
  - Loading：第一版数据本地加载很快，暂不做独立 Loading。
  - Empty：没有地点时显示“暂无地点数据”。
  - Error：数据读取异常时显示“地点数据加载失败”。
  - Success：展示地点列表。
- 验收标准：至少显示 2 个地点卡片。

## P002：详情页

- 页面目标：展示地点详情，并允许用户收藏。
- 页面入口：首页地点卡片。
- 主要模块：地点名称、地区、标签、说明、注意事项、收藏按钮。
- 用户动作：收藏 / 取消收藏、返回首页。
- 状态：
  - Empty：地点不存在时显示“地点不存在”。
  - Success：展示地点详情。
- 验收标准：点击地点后能看到详情，收藏按钮可用。

## P003：收藏页

- 页面目标：展示已收藏地点。
- 页面入口：顶部导航“收藏”。
- 主要模块：收藏列表、空状态。
- 用户动作：查看收藏地点、点击进入详情。
- 状态：
  - Empty：暂无收藏。
  - Success：展示已收藏地点。
- 验收标准：收藏后能在此页面看到对应地点。

## P004：设置页

- 页面目标：提供清空收藏和基础说明。
- 页面入口：顶部导航“设置”。
- 主要模块：清空收藏按钮、说明文案。
- 用户动作：清空收藏。
- 验收标准：点击清空后收藏页为空。

## P005：关于页

- 页面目标：说明这是课程实战样例。
- 页面入口：顶部导航“关于”。
- 主要模块：课程说明、技术边界、后续扩展方向。
- 验收标准：用户能明确知道当前是 H5 样例，不是完整商业产品。
