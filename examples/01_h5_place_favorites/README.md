# H5 实战样例：钓鱼露营地点收藏

这是《AI 产品工程实战》的第一版实战样例工程。

## 为什么第一版先做 H5

相比微信小程序，H5 的启动门槛更低：

- 不需要 AppID。
- 不需要微信开发者工具。
- 浏览器即可运行。
- 方便非技术学员理解页面、数据、交互和本地存储。
- 后续可以迁移到微信小程序、iOS App 或 Web App。

## 第一版页面

- 首页：展示地点列表
- 详情页：展示地点详情和收藏按钮
- 收藏页：展示已收藏地点
- 设置页：清空收藏和基础说明
- 关于页：说明课程样例

## 第一版不做

- 登录
- 后端
- 地图
- 发布
- 评论
- 私信
- 支付
- 云同步

## 如何运行

进入本目录后，用浏览器打开：

```text
index.html
```

如果浏览器限制本地文件访问，可以使用本地静态服务器：

```bash
python -m http.server 5173
```

然后访问：

```text
http://localhost:5173
```

## 核心流程

```text
首页浏览地点
↓
点击地点进入详情
↓
点击收藏
↓
进入收藏页查看
```

## 技术说明

第一版使用：

- 原生 HTML
- 原生 CSS
- 原生 JavaScript
- Mock 数据
- localStorage 本地收藏

不引入框架，降低非技术学员理解成本。
