# 非部署生产级检查清单

## 1. 产品与范围

- [ ] 产品定义已确认
- [ ] 第一版做什么已明确
- [ ] 第一版不做什么已明确
- [ ] 成功标准可验证

## 2. 设计与体验

- [ ] 界面规格已确认
- [ ] 参考截图或参考风格已记录
- [ ] 空状态、错误状态、加载状态已定义
- [ ] 多端体验边界已明确

## 3. 工程模板

- [ ] 已选择对应工程模板
- [ ] 没有让 AI 从 0 搭工程
- [ ] 模板基础结构保留
- [ ] 统一请求封装保留
- [ ] 统一错误处理保留
- [ ] 设计变量或主题配置保留

## 4. 接口与数据

- [ ] 接口契约已确认
- [ ] 数据库结构已确认
- [ ] 请求字段和响应字段可追溯
- [ ] 错误码可追溯
- [ ] 持久化字段可追溯

## 5. 安全

- [ ] 登录态策略已定义
- [ ] token 或 session 有有效期
- [ ] 输入参数有校验
- [ ] 不返回敏感字段
- [ ] 不提交真实密钥和密码
- [ ] CORS 策略已说明

## 6. 测试

- [ ] 前端核心交互有测试任务
- [ ] 后端业务逻辑有测试任务
- [ ] 接口联调有验证任务
- [ ] 失败场景有测试或验收说明

## 7. 验收

- [ ] 代码任务包已完成
- [ ] 工程模板检查清单已通过
- [ ] 代码交付检查清单已通过
- [ ] 使用验收已通过
- [ ] 变更记录已追加

## 8. 结论

全部通过后，可以称为“接近生产级的非部署交付”。

如果未通过，只能称为“工程化 MVP”或“Demo”。
