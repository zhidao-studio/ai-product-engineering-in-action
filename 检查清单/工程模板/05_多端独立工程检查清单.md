# 多端独立工程检查清单

## 1. 工程边界

- [ ] Web 后台是独立工程
- [ ] H5 是独立工程
- [ ] 微信小程序是独立工程
- [ ] iOS 是独立工程
- [ ] 没有建立前端共享层

## 2. 一致性来源

- [ ] 多端接口来自同一份接口契约
- [ ] 多端字段来自同一份字段定义
- [ ] 多端错误码来自同一份接口契约
- [ ] 多端体验分别由各端界面规格约束

## 3. 禁止混用

- [ ] Web 后台没有引用 H5 代码
- [ ] Web 后台没有引用小程序代码
- [ ] H5 没有引用 Web 后台代码
- [ ] 小程序没有引用 Web/H5 页面组件
- [ ] iOS 没有复制 Web/H5 页面结构

## 4. 构建检查

- [ ] 每个端有独立安装方式
- [ ] 每个端有独立检查命令
- [ ] 每个端有独立测试方式
- [ ] 每个端有独立构建方式
