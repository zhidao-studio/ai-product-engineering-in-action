# 前端多端基座检查清单

## 1. 开源借鉴

- [ ] 已参考成熟项目的分层和构建体系
- [ ] 已明确借鉴来源
- [ ] 没有闭门自造多端架构
- [ ] 没有直接搬入大型平台复杂度

## 2. 多端边界

- [ ] Web 后台、H5、小程序、iOS 端能力边界清楚
- [ ] 页面和组件没有跨端互相直接引用
- [ ] Web 后台没有引用 H5 / 小程序端专属能力
- [ ] H5 没有引用 Web 后台 / 小程序端专属能力
- [ ] 小程序没有引用 Web / H5 页面组件
- [ ] iOS 没有复制 Web / H5 页面结构

## 3. 共享层

- [ ] API 类型可共享
- [ ] 业务模型可共享
- [ ] 错误码可共享
- [ ] 设计 token 可共享
- [ ] 纯函数工具可共享
- [ ] 共享层不包含端专属组件
- [ ] 共享层变化已检查所有端影响

## 4. 构建体系

- [ ] 每个端都有 lint 命令
- [ ] 每个端都有 typecheck 或等价检查
- [ ] 每个端都有 test 或测试计划
- [ ] 每个端都有 build 或构建说明
- [ ] 每个端都有 check 汇总命令或检查步骤

## 5. AI 修改边界

- [ ] 修改单端页面时没有改其他端页面
- [ ] 修改共享类型前先更新接口契约
- [ ] 修改设计 token 时说明影响端
- [ ] 未经任务包批准没有新增跨端框架
- [ ] 未经任务包批准没有把原生端改成跨端方案
