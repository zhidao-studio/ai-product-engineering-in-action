# 前端检查清单

## 1. 技术栈

- [ ] 使用 Vue 3
- [ ] 使用 Vite
- [ ] 使用 TypeScript
- [ ] 使用 Vue Router
- [ ] 状态管理使用 Pinia
- [ ] 使用 ESLint 和 Prettier

## 2. 目录结构

- [ ] 页面放在 `src/pages/`
- [ ] 组件放在 `src/components/`
- [ ] API 调用放在 `src/services/`
- [ ] 请求和错误拦截放在统一位置
- [ ] 状态放在 `src/stores/`
- [ ] 类型放在 `src/types/`
- [ ] 样式放在 `src/styles/`

## 3. 编码约束

- [ ] 页面中没有直接散落 fetch
- [ ] API 地址没有硬编码
- [ ] 字段来自数据规范
- [ ] 页面状态包含 loading / empty / success / error
- [ ] 样式符合设计规格
- [ ] 只有跨页面共享状态进入 store
- [ ] 页面内临时状态没有被无差别放入 store

## 4. 错误处理

- [ ] API 错误经过统一拦截或统一请求封装
- [ ] 401 行为符合接口契约
- [ ] 网络错误展示用户可理解提示
- [ ] 表单校验错误在字段附近或表单区域展示
- [ ] 不向用户展示原始异常堆栈
- [ ] 没有每个接口各写一套不同错误处理逻辑

## 5. 验证

- [ ] `npm install` 可执行
- [ ] `npm run dev` 可启动
- [ ] `npm run build` 可通过
- [ ] 关键页面可访问
- [ ] 关键交互可完成
- [ ] 前端核心交互有测试任务或测试说明
