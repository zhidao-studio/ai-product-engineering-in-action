# 微信云托管适配检查清单

## 1. 工程边界

- [ ] Web 后台是独立工程
- [ ] H5 是独立工程
- [ ] 微信小程序是独立工程
- [ ] 后端是 Spring Boot 单体应用
- [ ] 没有为了部署引入 Spring Cloud
- [ ] 没有为了部署合并多端前端工程

## 2. Web / H5 静态服务

- [ ] 构建命令已明确
- [ ] 产物目录已明确
- [ ] API 地址通过配置注入
- [ ] 没有在页面里写死后端域名

## 3. 后端容器服务

- [ ] 后端存在 Dockerfile 或容器构建说明
- [ ] 后端启动端口已明确
- [ ] 后端提供健康检查接口
- [ ] 配置通过环境变量注入
- [ ] 没有提交真实密钥、数据库密码或微信密钥

## 4. 微信小程序接入

- [ ] 小程序后端地址通过配置管理
- [ ] 小程序接口路径以接口契约为准
- [ ] 云托管访问方式以当前控制台为准

## 5. 文档确认

- [ ] 已阅读微信云托管适配方案
- [ ] 当前项目确实选择微信云托管作为部署目标
- [ ] 当前配置以微信云托管当前控制台和官方文档为准
