# 页面原型确认清单

## 1. 目标

本清单用于确认页面原型是否已经达到可以进入正式开发的程度。

没有完成页面原型确认，不应进入工程模板复制和编码阶段。

## 2. 文件确认

- [ ] 已生成页面原型文件
- [ ] 已生成原型确认记录
- [ ] 原型可以直接打开查看
- [ ] 原型不依赖真实后端
- [ ] 原型使用模拟数据

## 3. 页面确认

- [ ] 页面方向符合产品定义
- [ ] 页面布局符合界面规格
- [ ] 信息层级清楚
- [ ] 主入口位置正确
- [ ] 关键按钮位置正确
- [ ] 表单区域位置正确
- [ ] 列表和详情区域符合预期

## 4. 交互确认

- [ ] 可以切换主要页面或区域
- [ ] 可以点击主要按钮
- [ ] 可以填写主要表单
- [ ] 可以看到操作反馈
- [ ] 可以查看空状态
- [ ] 可以查看错误状态
- [ ] 可以查看加载状态

## 5. 边界确认

- [ ] 原型代码不会复制到正式工程中
- [ ] 原型不替代工程模板
- [ ] 原型只用于视觉和交互确认
- [ ] 修改意见已记录
- [ ] 使用者已确认可以进入工程阶段

## 6. 结论

全部通过后，才能进入：

```text
工程规格
复制工程模板
生成代码任务包
AI 编码
```
