# 检查清单

本目录存放可执行检查清单。

检查清单用于判断当前文档、原型、工程模板、代码任务包或代码交付是否已经满足进入下一步的条件。

## 1. 目录说明

```text
检查清单/约束框架/    约束框架完整性检查
检查清单/代码交付/    页面原型、前端、后端、接口、数据库、基础设施、云托管适配检查
检查清单/工程模板/    Web、H5、微信小程序、iOS、后端和多端独立工程检查
检查清单/生产级/      非部署生产级交付边界检查
```

## 2. 使用顺序

```text
产品定义和界面规格完成后：
  检查清单/约束框架/

高保真交互预览完成后：
  检查清单/代码交付/05_页面原型确认清单.md

复制工程模板前：
  检查清单/工程模板/

生成代码任务包后：
  检查清单/代码交付/

后端或数据库变更后：
  检查清单/代码交付/03_后端检查清单.md
  文档/工程基线/07_数据库设计规范.md
  文档/工程基线/08_码值规范.md

接近生产级非部署交付前：
  检查清单/生产级/
```

## 3. 原则

```text
先检查清单，后行动。
先判断，后修改。
不通过时先补文档、原型、任务包或缩小任务范围。
检查清单不替代真实运行验证。
```
