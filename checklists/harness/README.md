# Harness Checklist

本目录提供可直接使用的检查清单。

Checklist 的作用是让非技术人也能判断：

```text
当前 harness 是否完整？
AI 是否在约束内执行？
产物是否可以进入下一步？
```

## 清单列表

| 文件 | 作用 |
|---|---|
| `01_HARNESS_COMPLETENESS_CHECKLIST.md` | 检查 harness 是否完整 |
| `02_NON_TECH_USER_CHECKLIST.md` | 非技术人使用前后的检查清单 |

## 使用原则

```text
先 checklist，后执行。
先判断，后修改。
先补 harness，再让 AI 继续。
```
