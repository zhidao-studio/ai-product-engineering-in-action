# 非技术人使用检查清单

## 1. 让 AI 执行前

- [ ] 我知道这个产品给谁用
- [ ] 我知道第一版只做什么
- [ ] 我知道第一版不做什么
- [ ] 我能描述页面大概长什么样
- [ ] 我能说清核心流程
- [ ] 我知道 AI 这次只做哪个任务
- [ ] 我知道 AI 可以改哪些文件或内容
- [ ] 我知道 AI 不应该改哪些内容
- [ ] 我知道做完后怎么判断通过

## 2. AI 执行后

- [ ] AI 说明了改了什么
- [ ] AI 说明了对应哪份文档
- [ ] AI 没有明显扩大范围
- [ ] 结果能对应到任务
- [ ] 结果能对应到验收标准
- [ ] 我能不看代码判断是否基本正确

## 3. 不通过时

先判断：

```text
是 harness 没写清楚？
还是 AI 没按 harness 执行？
```

如果 harness 没写清楚：

- [ ] 先补模板或 checklist
- [ ] 再重新生成任务
- [ ] 不直接手工改代码

如果 AI 没按 harness 执行：

- [ ] 缩小任务范围
- [ ] 明确允许和限制范围
- [ ] 重新给 AI 执行

## 4. 最终判断

```text
是否可以进入下一步：是 / 否
原因：
需要补哪份 harness：
```
