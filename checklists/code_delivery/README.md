# Code Delivery 检查清单

本目录用于检查从 0 开始的代码工程是否具备可交给 AI 编码的工程基线。

## 清单列表

| 文件 | 作用 |
|---|---|
| `01_ENGINEERING_BASELINE_CHECKLIST.md` | 工程基线完整性检查 |
| `02_FRONTEND_CHECKLIST.md` | Vue 前端工程检查 |
| `03_BACKEND_CHECKLIST.md` | Spring Boot 后端工程检查 |
| `04_API_CHECKLIST.md` | API 契约检查 |
| `05_INFRA_CHECKLIST.md` | macOS / Homebrew / nginx / ngrok 基础设施检查 |

## 使用原则

```text
没有工程基线，不交给 AI 写代码。
没有代码任务包，不允许 AI 修改工程。
没有代码验收，不算代码交付完成。
```
