# 文档导航

本目录只放说明性文档。

样板、检查清单、指令、案例分别放在仓库顶层对应目录，不放进文档目录。

## 1. 目录职责

```text
文档/仓库结构说明.md        仓库结构说明
文档/方法论/                方法论：目标、原则、分层、构造方法、覆盖检查
文档/工程基线/              工程基线：技术选择、分层规则、质量红线
文档/教学化内容/            教学化内容：后续面向非技术人的课程化表达
```

## 2. 阅读顺序

```text
1. 文档/仓库结构说明.md
2. 文档/方法论/01_目标与边界.md
3. 文档/方法论/02_核心原则.md
4. 文档/方法论/03_约束框架分层结构.md
5. 文档/方法论/04_如何构造约束框架.md
6. 文档/方法论/05_覆盖检查.md
7. 文档/工程基线/01_工程基线总说明.md
8. 文档/工程基线/02_工程质量红线.md
```

## 3. 关联目录

