# Agent 修复问题提示词

请根据以下报错或问题修复当前任务。

## 核心原则

只修复当前问题，不做无关重构，不新增未经文档批准的功能。

## 必须阅读

- 当前任务说明
- 相关产品文档
- 文档到代码映射
- 当前报错或问题说明

## 禁止修改

- 不改无关文件
- 不新增功能
- 不引入新依赖
- 不改变数据字段
- 不改变页面范围
- 不重构当前任务之外的代码

## 完成后输出

1. 问题根因
2. 修改文件列表
3. 修改内容说明
4. 这次修改对应哪份文档或哪个任务
5. 验证方式
6. 风险点

## 自检问题

```text
是否只修复当前问题？
是否没有扩大需求范围？
是否仍然符合文档到代码映射？
是否满足当前任务验收标准？
```
