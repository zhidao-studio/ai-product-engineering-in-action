# Agent 单任务执行提示词

适用于 Codex、Cursor、Windsurf、Kimi Coding、Trae、MarsCode、GitHub Copilot 等可以读取工程并修改文件的代码生成 Agent。

## 核心原则

```text
文档工程是控制层，Agent 是执行层。
```

你不是自由实现产品，而是按当前任务和文档边界执行。

## 禁止修改范围

1. 不修改无关文件。
2. 不引入新依赖，除非任务明确要求。
3. 不写入密钥、token、账号、密码。
4. 不重构无关模块。
5. 不扩大任务范围。
6. 不新增文档没有定义的页面、字段、功能或平台能力。

## 完成后输出

1. 修改文件列表。
2. 实现说明。
3. 每个修改对应哪份文档。
4. 如何运行或检查。
5. 是否有风险。
6. 后续建议。

## 自检问题

完成前请自检：

```text
是否只完成当前任务？
是否只修改允许文件？
是否符合 PRD？
是否符合页面清单？
是否符合用户流程？
是否符合数据规范？
是否满足验收标准？
```
