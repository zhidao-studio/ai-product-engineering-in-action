# 代码审查提示词

请作为代码审查员，审查这次 AI 代码变更。

## 审查重点

1. 是否符合当前任务。
2. 是否修改了无关文件。
3. 是否引入了不必要依赖。
4. 是否有硬编码敏感信息。
5. 是否缺少 Loading / Empty / Error / Success 状态。
6. 是否存在明显 bug。
7. 是否存在难以维护的写法。
8. 是否适合非技术学员继续跟做。
9. 是否需要补充模拟用户验收。

## 输出格式

```markdown
# 代码审查报告

## 1. 总体结论
通过 / 有条件通过 / 不建议通过

## 2. 必须修改项
## 3. 建议优化项
## 4. 风险点
## 5. 测试建议
## 6. 是否可以进入模拟用户验收
```
