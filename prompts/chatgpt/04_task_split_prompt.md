# 任务拆分提示词

请根据项目文档，把项目拆成 AI 编码工具可以执行的小任务。

## 要求

1. 一个任务只做一件事。
2. 每个任务必须能独立验收。
3. 每个任务必须写允许修改范围。
4. 每个任务必须写禁止修改范围。
5. 每个任务必须有验收标准。
6. 任务数量控制在 5-12 个。
7. 如果任务太大，请继续拆小。

## 输出格式

```markdown
# AI 任务清单

| 任务编号 | 任务名称 | 依赖 | 允许修改范围 | 禁止修改范围 | 验收标准 | 状态 |
|---|---|---|---|---|---|---|
```

## 输入

【粘贴项目总说明、PRD、页面清单】
