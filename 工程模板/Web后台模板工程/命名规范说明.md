# Web 后台命名规范说明

本工程必须遵守：

```text
文档/工程基线/06_前后端命名规范.md
```

## 1. 页面

页面组件使用 PascalCase，并以 Page 结尾：

```text
LoginPage.vue
DashboardPage.vue
VideoTaskListPage.vue
VideoTaskDetailPage.vue
```

## 2. 组件

业务组件使用业务名 + 组件类型：

```text
VideoTaskForm.vue
VideoTaskTable.vue
VideoTaskCard.vue
```

状态组件使用：

```text
LoadingState.vue
EmptyState.vue
ErrorState.vue
```

## 3. Service

API 调用文件使用业务名 + Service：

```text
sessionService.ts
videoTaskService.ts
```

函数使用动词 + 业务对象：

```text
login
listVideoTasks
getVideoTask
createVideoTask
updateVideoTask
```

## 4. Store

Store 文件使用业务名 + Store：

```text
userStore.ts
videoTaskStore.ts
```

Store hook 使用：

```text
useUserStore
useVideoTaskStore
```

## 5. 类型

请求类型以 Request 结尾。

响应类型以 Response 结尾。

业务模型使用 Model 或 ViewModel。

## 6. 禁止

不得为同一业务概念新增多个英文名。

例如已有 `VideoTask`，不得再新增：

```text
VideoJob
TaskVideo
VideoMission
```
