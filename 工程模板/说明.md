# 工程模板

本目录存放可直接复制使用的前端和后端模板工程。

工程模板不是文档样板，而是真实可运行的工程骨架。

## 目录

```text
工程模板/前端模板工程/    Vue 3 + Vite + TypeScript + Element Plus 轻量后台模板
工程模板/后端模板工程/    Spring Boot 单体轻量后端模板
```

## 使用原则

```text
先写项目约束文档，
再复制工程模板，
再让 AI 在模板工程内按任务包开发。
```

AI 不应从 0 创建工程，也不应删除模板工程中的基础结构、统一错误处理、统一响应、检查脚本和测试样例。

## 和其他目录的关系

```text
文档/        定义方法、操作流程和工程基线
样板/        定义文档和任务包怎么写
工程模板/    提供真实可运行的工程骨架
检查清单/    检查工程和任务是否合格
指令/        约束 AI 如何生成文档和执行任务
案例/        展示完整项目如何串联
```
