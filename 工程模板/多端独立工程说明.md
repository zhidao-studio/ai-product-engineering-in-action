# 多端独立工程说明

Web 后台、H5、微信小程序、iOS 是多个独立工程。

不建立前端共享层。

多端一致性通过接口契约、产品定义、界面规格、工程规格、代码任务包和检查清单保持。

每个端都有自己的工程结构、构建命令、API 封装、类型定义、设计变量、页面组件和测试方式。

参考方向：

```text
Web 后台：vue-element-plus-admin / Vue Vben Admin
H5：Vant
微信小程序：Vant Weapp / Taro
iOS：SwiftUI / TCA / Kingfisher
```

原则：

```text
多个工程，独立建设。
接口一致，代码不共享。
各端体验，各端负责。
```
