# 通用大模型指令

本目录放通用大模型指令。

用途：

- 逐层起草约束框架
- 完善样板
- 生成检查清单
- 生成案例

## 指令清单

| 文件 | 用途 |
|---|---|
| `01_构造约束框架指令.md` | 生成逐层构造计划，不一次性生成 7 层 |
| `02_产品定义约束指令.md` | 只起草产品定义约束 |
| `03_界面规格约束指令.md` | 基于产品定义起草界面规格约束 |
| `04_工程规格约束指令.md` | 基于产品定义和界面规格起草工程规格约束 |
| `05_接口与数据约束指令.md` | 基于工程规格起草接口契约和数据库结构 |
| `06_任务包约束指令.md` | 基于已确认文档拆分代码任务包 |
| `07_验证约束指令.md` | 基于任务包起草验证约束 |
| `08_使用验收与交付迭代指令.md` | 起草使用验收和交付迭代约束 |

## 使用原则

```text
逐层生成。
逐层确认。
上一层未确认，不进入下一层。
```
