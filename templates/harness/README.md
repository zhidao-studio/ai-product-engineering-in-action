# Harness 模板

本目录提供可复用的文档模板。

模板目标不是写漂亮文档，而是让 AI 产品交付过程可约束、可验收、可追溯。

## 模板清单

| 模板 | 作用 |
|---|---|
| `01_PRODUCT_DEFINITION_HARNESS.md` | 产品定义 Harness |
| `02_DESIGN_SPEC_HARNESS.md` | 设计规格 Harness |
| `03_ENGINEERING_SPEC_HARNESS.md` | 工程规格 Harness |
| `04_TASK_EXECUTION_HARNESS.md` | 任务执行 Harness |
| `05_VALIDATION_HARNESS.md` | 验证 Harness |
| `06_ACCEPTANCE_HARNESS.md` | 体验验收 Harness |
| `07_DELIVERY_ITERATION_HARNESS.md` | 交付迭代 Harness |

## 使用方式

1. 复制模板。
2. 替换为你的项目内容。
3. 交给 AI / Agent 之前先跑 checklist。
4. AI 交付后用对应 checklist 验收。

## 原则

```text
模板不是记录结果，
而是约束 AI 接下来能做什么。
```
