# 用户流程模板

这份文档用于控制用户动作和结果闭环。

## 1. 主流程

```text
用户进入
↓
看到内容
↓
进入详情
↓
执行核心动作
↓
看到动作结果
```

## 2. 页面跳转

| 来源页面 | 用户动作 | 目标页面 / 结果 | 说明 |
|---|---|---|---|

## 3. 动作结果检查

| 用户动作 | 必须看到的结果 | 是否闭环 |
|---|---|---|

## 4. 反面流程

请列出可能断掉的流程：

```text
用户做了动作，但看不到结果：
```

## 5. 流程控制检查

- [ ] 每个用户动作都有结果。
- [ ] 核心流程能闭环。
- [ ] 页面跳转来自页面清单。
- [ ] 没有新增 PRD 之外的流程。
- [ ] Agent 不得自行发明跳转关系。
