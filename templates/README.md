# 模板

本目录存放可复用模板。

当前主线模板只放在：

```text
templates/harness/
```

## 目录说明

| 目录 | 作用 |
|---|---|
| `harness/` | 七层 AI 产品交付 Harness 模板 |

## 使用方式

1. 先阅读 `examples/01_minimal_harness/`，看填好的示例。
2. 再复制 `templates/harness/` 下的模板。
3. 替换成自己的项目内容。
4. 用 `checklists/harness/` 检查是否完整。

## 原则

```text
模板不是文档存档，
而是约束 AI 接下来能做什么。
```
