# 模板

本目录存放可复用模板。

## 目录说明

```text
templates/harness/        七层 AI 产品交付 Harness 模板
templates/code_delivery/  工程基线、前后端规范、API、数据库、基础设施和任务包模板
```

## 使用方式

1. 先看 `examples/01_minimal_harness/`，理解一套填好的 harness。
2. 再使用 `templates/harness/` 定义产品、设计、工程和验收约束。
3. 需要进入代码交付时，再使用 `templates/code_delivery/` 定义工程基线和任务包。
4. 最后用 `checklists/` 检查完整性。
