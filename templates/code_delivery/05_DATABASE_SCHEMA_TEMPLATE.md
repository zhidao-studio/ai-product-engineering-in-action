## 5. 时间字段规范

默认字段：

```text
created_at
updated_at
```

如需软删除：

```text
deleted_at
```

## 6. 数据迁移规范

迁移文件位置：

```text
backend/src/main/resources/db/migration/
```

推荐命名：

```text
V001__init_schema.sql
V002__add_xxx_table.sql
```

## 7. AI 禁止行为

AI 不得：

- 自行新增表
- 自行改变字段含义
- 自行删除字段
- 自行绕过迁移文件直接改库
- 自行新增复杂权限模型

## 8. 验收标准

- [ ] 表结构来自数据规范
- [ ] 字段类型明确
- [ ] 必填和默认值明确
- [ ] 索引和唯一约束明确
- [ ] 迁移文件可追溯
