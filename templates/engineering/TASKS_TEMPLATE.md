# 任务清单模板

这份文档用于控制 Agent 每次执行什么、允许改什么、禁止改什么、如何验收。

## 1. 状态说明

| 状态 | 含义 |
|---|---|
| TODO | 未开始 |
| DOING | 正在做 |
| REVIEW | 等待审查 |
| FIXING | 修复中 |
| DONE | 已完成 |

## 2. 任务清单

| 任务编号 | 任务名称 | 依赖文档 | 允许修改范围 | 禁止修改范围 | 验收标准 | 状态 |
|---|---|---|---|---|---|---|
| TASK-001 |  |  |  |  |  | TODO |

## 3. Agent 输入模板

```text
当前任务：TASK-xxx

目标：

必须阅读：

允许修改：

禁止修改：

验收标准：

完成后输出：
1. 修改文件列表
2. 实现说明
3. 对应文档
4. 验证方式
5. 风险点
```

## 4. 任务控制检查

- [ ] 一个任务只做一件事。
- [ ] 每个任务都有依赖文档。
- [ ] 每个任务都有允许修改范围。
- [ ] 每个任务都有禁止修改范围。
- [ ] 每个任务都有验收标准。
- [ ] Agent 不得自行合并多个任务。
