# 接口规范模板

这份文档用于控制前端和后端之间的数据边界。

如果第一版不做真实接口，也要用本文件说明 Mock 数据如何模拟接口，以及后续如何迁移。

## 1. 接口结论

第一版是否需要真实 API：是 / 否

第一版数据方案：

```text
Mock 数据 / localStorage / 静态 JSON / API
```

## 7. 联调验收清单

- [ ] 前端需要的字段都在接口或 Mock 中。
- [ ] 接口字段和数据规范一致。
- [ ] Empty / Error / Success 状态都有定义。
- [ ] Mock 可以迁移到真实接口。
- [ ] 没有新增 PRD 之外的接口能力。
