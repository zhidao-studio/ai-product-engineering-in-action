# 工程规格

## 1. 技术基线

引用默认工程基线：

```text
文档/工程基线/01_工程基线总说明.md
文档/工程基线/02_工程质量红线.md
```

## 2. 前端范围

前端使用 Vue 3 + Vite + TypeScript。

范围：

- 登录页
- 任务列表页
- 新建任务页
- 任务详情页
- API 调用封装
- 登录状态本地保存
- loading / empty / success / error 状态

## 3. 后端范围

后端使用 Java + Spring Boot 单体应用。

范围：

- 登录接口
- 任务查询接口
- 任务创建接口
- 任务详情接口
- 任务更新接口
- 统一错误响应
- 参数校验

## 4. 数据范围

核心模型：

- 用户
- 视频任务

第一版只支持单用户，不做复杂角色权限。

## 5. 目录边界

目标项目结构：

```text
project-root/
  docs/
  frontend/
  backend/
  infra/
  scripts/
  README.md
```

AI 不得新增未定义顶层目录。

## 6. 技术限制

- 不使用 Spring Cloud
- 不引入微服务
- 不接真实视频模型
- 不暴露数据库公网端口
- 不把密钥写入代码
- 不把接口地址硬编码到页面里
