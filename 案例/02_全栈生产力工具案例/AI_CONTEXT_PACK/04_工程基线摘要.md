# 工程基线摘要

本案例遵循当前仓库工程基线。

## 1. 前端

```text
Web 后台：Vue 3 + Vite + TypeScript + Pinia + Element Plus
H5：Vue 3 + Vite + TypeScript + Pinia + Vant
```

## 2. 后端

```text
Java 17+
Spring Boot 3
Maven
MyBatis-Plus
Flyway
单体应用
不使用 Spring Cloud
```

## 3. 数据库

```text
MySQL 优先
主键使用 CHAR(32)
每张业务表默认 7 要素
时间字段由 trigger 维护
```

## 4. iOS

```text
SwiftUI
MVVM
CocoaPods
XCTest
```

不得默认使用 Swift Package Manager。
