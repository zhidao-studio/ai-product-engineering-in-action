# 界面规格

## 1. 页面清单

| 页面 | 用途 |
|---|---|
| 登录页 | 输入账号密码进入后台 |
| 任务列表页 | 查看全部视频任务和状态 |
| 新建任务页 | 创建视频任务 |
| 任务详情页 | 查看和编辑任务内容 |

## 2. 视觉风格

- 简洁后台风格
- 左侧导航 + 主内容区
- 卡片式任务列表
- 状态使用标签展示
- 表单字段清楚，避免复杂布局

## 3. 页面状态

每个核心页面必须包含：

- 加载中
- 空状态
- 成功状态
- 错误状态

## 4. 模块说明

### 登录页

- 标题
- 账号输入框
- 密码输入框
- 登录按钮
- 错误提示

### 任务列表页

- 页面标题
- 新建任务按钮
- 状态筛选
- 任务卡片或表格
- 空状态提示

### 新建任务页

- 任务标题
- 视频主题
- 脚本摘要
- 分镜摘要
- 保存按钮

### 任务详情页

- 基础信息
- 状态切换
- 脚本摘要
- 分镜摘要
- 保存按钮

## 5. 设计输入要求

如果进入真实项目，用户应补充：

- 一个喜欢的后台系统截图
- 一个不喜欢的反面截图
- 是否偏飞书/Notion/企业后台风格

没有截图时，AI 只能按“简洁内部后台”处理，不能自行做强风格设计。
