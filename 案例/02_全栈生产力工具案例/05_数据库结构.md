# 数据库结构

## 1. 数据库选择

第一版使用 PostgreSQL 或 MySQL 均可。

本案例默认使用 PostgreSQL。

## 2. 数据模型

| 模型 | 表名 | 说明 |
|---|---|---|
| 用户 | users | 登录用户 |
| 视频任务 | video_tasks | AI 视频生产任务 |

## 3. users 表

| 字段 | 类型 | 是否必填 | 约束 | 说明 |
|---|---|---|---|---|
| id | varchar(64) | 是 | 主键 | 用户 ID |
| username | varchar(64) | 是 | 唯一 | 登录账号 |
| password_hash | varchar(255) | 是 |  | 密码哈希 |
| display_name | varchar(100) | 是 |  | 显示名称 |
| created_at | timestamp | 是 |  | 创建时间 |
| updated_at | timestamp | 是 |  | 更新时间 |

## 4. video_tasks 表

| 字段 | 类型 | 是否必填 | 约束 | 说明 |
|---|---|---|---|---|
| id | varchar(64) | 是 | 主键 | 任务 ID |
| title | varchar(200) | 是 |  | 任务标题 |
| topic | varchar(200) | 是 |  | 视频主题 |
| script_summary | text | 否 |  | 脚本摘要 |
| storyboard_summary | text | 否 |  | 分镜摘要 |
| status | varchar(32) | 是 |  | draft / pending / generating / completed |
| created_by | varchar(64) | 是 | 外键 users.id | 创建人 |
| created_at | timestamp | 是 |  | 创建时间 |
| updated_at | timestamp | 是 |  | 更新时间 |

## 5. 索引和约束

| 表 | 字段 | 类型 | 说明 |
|---|---|---|---|
| users | username | 唯一索引 | 登录账号唯一 |
| video_tasks | status | 普通索引 | 支持按状态筛选 |
| video_tasks | created_at | 普通索引 | 支持按创建时间排序 |

## 6. 迁移文件

推荐位置：

```text
backend/src/main/resources/db/migration/
```

推荐文件：

```text
V001__init_users_and_video_tasks.sql
```

## 7. 安全要求

- 不存明文密码
- 密码字段只保存哈希值
- 不在接口中返回 password_hash
- 后续如支持多用户，需要补权限模型
