# 接口契约

## 1. 基础约定

接口前缀：

```text
/api/v1
```

响应格式：

```json
{
  "code": "OK",
  "message": "success",
  "data": {}
}
```

错误格式：

```json
{
  "code": "ERROR_CODE",
  "message": "可读错误说明",
  "data": null
}
```

## 2. API 版本规则

第一版所有接口使用 `/api/v1`。

不兼容变更必须新增 `/api/v2`，不得直接破坏 v1 的请求字段、响应字段、错误码和语义。

## 3. 多端接入规则

本后端默认支持多端接入：

```text
Web 后台
H5
微信小程序
iOS
```

可选请求头：

```text
X-Client-Type: web-admin / h5 / weapp / ios
```

客户端类型用于日志、审计、兼容判断和灰度判断，不用于在业务逻辑中大量散落端类型条件分支。

## 4. 认证规则

第一版采用单用户登录。

登录态策略：

```text
优先使用服务端设置的 HttpOnly Cookie。
Cookie 需要设置 SameSite。
如需跨源携带凭证，前端请求和后端 CORS 必须同时明确 credentials 策略。
```

token 或 session 必须有有效期。

401 响应后，前端必须提示登录失效并跳转登录页。

登录失败统一返回：

```text
账号或密码错误
```

不得区分提示“用户不存在”或“密码错误”。

## 6. 登录接口

### POST `/api/v1/session`

请求头：

```text
X-Client-Type: web-admin / h5 / weapp / ios
```

请求：

```json
{
  "username": "admin",
  "password": "password"
}
```

响应：

```json
{
  "code": "OK",
  "message": "success",
  "data": {
    "user": {
      "id": "string",
      "username": "string",
      "displayName": "string"
    }
  }
}
```

说明：登录成功后，由服务端设置登录 Cookie 或按认证策略返回登录凭证，响应体不返回敏感字段。

## 7. 视频任务字段

| 字段 | 类型 | 是否必填 | 说明 | 兼容要求 |
|---|---|---|---|---|
| id | string | 是 | 任务 ID | 不得删除 |
| title | string | 是 | 任务标题 | 不得删除 |
| topic | string | 是 | 视频主题 | 不得删除 |
| scriptSummary | string | 否 | 脚本摘要 | 可为空 |
| storyboardSummary | string | 否 | 分镜摘要 | 可为空 |
| status | string | 是 | draft / pending / generating / completed | 新状态需向后兼容 |
| createdAt | string | 是 | 创建时间 | 不得删除 |
| updatedAt | string | 是 | 更新时间 | 不得删除 |

## 8. 创建任务接口

### POST `/api/v1/video-tasks`

请求：

```json
{
  "title": "2026 俄罗斯留学申请时间线",
  "topic": "俄罗斯留学申请",
  "scriptSummary": "介绍申请节奏和关键时间点",
  "storyboardSummary": "按时间轴展示"
}
```

响应：返回视频任务对象。

## 9. 更新任务接口

### PUT `/api/v1/video-tasks/{id}`

请求：

```json
{
  "title": "string",
  "topic": "string",
  "scriptSummary": "string",
  "storyboardSummary": "string",
  "status": "draft"
}
```

响应：返回更新后的视频任务对象。

## 10. 错误码

| 错误码 | HTTP 状态 | 说明 | 前端行为 |
|---|---:|---|---|
| INVALID_REQUEST | 400 | 请求参数错误 | 展示字段或表单错误 |
| UNAUTHORIZED | 401 | 未登录或登录失效 | 提示登录失效并跳转登录页 |
| NOT_FOUND | 404 | 资源不存在 | 展示资源不存在提示 |
| INTERNAL_ERROR | 500 | 服务内部错误 | 展示系统错误提示 |
