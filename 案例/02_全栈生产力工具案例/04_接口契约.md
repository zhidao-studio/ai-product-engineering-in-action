# 接口契约

## 1. 基础约定

接口前缀：

```text
/api
```

响应格式：

```json
{
  "code": "OK",
  "message": "success",
  "data": {}
}
```

错误格式：

```json
{
  "code": "ERROR_CODE",
  "message": "可读错误说明",
  "data": null
}
```

## 2. 接口列表

| 编号 | 方法 | 路径 | 用途 | 调用页面 |
|---|---|---|---|---|
| API-001 | POST | `/api/auth/login` | 登录 | 登录页 |
| API-002 | GET | `/api/video-tasks` | 查询任务列表 | 任务列表页 |
| API-003 | POST | `/api/video-tasks` | 新建任务 | 新建任务页 |
| API-004 | GET | `/api/video-tasks/{id}` | 查询任务详情 | 任务详情页 |
| API-005 | PUT | `/api/video-tasks/{id}` | 更新任务 | 任务详情页 |

## 3. 登录接口

### POST `/api/auth/login`

请求：

```json
{
  "username": "admin",
  "password": "password"
}
```

响应：

```json
{
  "code": "OK",
  "message": "success",
  "data": {
    "token": "string",
    "user": {
      "id": "string",
      "username": "string",
      "displayName": "string"
    }
  }
}
```

## 4. 视频任务字段

| 字段 | 类型 | 是否必填 | 说明 |
|---|---|---|---|
| id | string | 是 | 任务 ID |
| title | string | 是 | 任务标题 |
| topic | string | 是 | 视频主题 |
| scriptSummary | string | 否 | 脚本摘要 |
| storyboardSummary | string | 否 | 分镜摘要 |
| status | string | 是 | draft / pending / generating / completed |
| createdAt | string | 是 | 创建时间 |
| updatedAt | string | 是 | 更新时间 |

## 5. 创建任务接口

### POST `/api/video-tasks`

请求：

```json
{
  "title": "2026 俄罗斯留学申请时间线",
  "topic": "俄罗斯留学申请",
  "scriptSummary": "介绍申请节奏和关键时间点",
  "storyboardSummary": "按时间轴展示"
}
```

响应：返回视频任务对象。

## 6. 更新任务接口

### PUT `/api/video-tasks/{id}`

请求：

```json
{
  "title": "string",
  "topic": "string",
  "scriptSummary": "string",
  "storyboardSummary": "string",
  "status": "draft"
}
```

响应：返回更新后的视频任务对象。

## 7. 错误码

| 错误码 | HTTP 状态 | 说明 |
|---|---:|---|
| INVALID_REQUEST | 400 | 请求参数错误 |
| UNAUTHORIZED | 401 | 未登录或登录失效 |
| NOT_FOUND | 404 | 资源不存在 |
| INTERNAL_ERROR | 500 | 服务内部错误 |
