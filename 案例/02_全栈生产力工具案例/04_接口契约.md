# 接口契约

## 1. 基础约定

接口前缀：

```text
/api
```

响应格式：

```json
{
  "code": "OK",
  "message": "success",
  "data": {}
}
```

错误格式：

```json
{
  "code": "ERROR_CODE",
  "message": "可读错误说明",
  "data": null
}
```

## 2. 认证规则

第一版采用单用户登录。

登录态策略：

```text
优先使用服务端设置的 HttpOnly Cookie。
Cookie 需要设置 SameSite。
如需跨源携带凭证，前端请求和后端 CORS 必须同时明确 credentials 策略。
```

token 或 session 必须有有效期。

401 响应后，前端必须提示登录失效并跳转登录页。

登录失败统一返回：

```text
账号或密码错误
```

不得区分提示“用户不存在”或“密码错误”。

## 3. 接口列表

| 编号 | 方法 | 路径 | 用途 | 调用页面 | 是否认证 |
|---|---|---|---|---|---|
| API-001 | POST | `/api/auth/login` | 登录 | 登录页 | 否 |
| API-002 | GET | `/api/video-tasks` | 查询任务列表 | 任务列表页 | 是 |
| API-003 | POST | `/api/video-tasks` | 新建任务 | 新建任务页 | 是 |
| API-004 | GET | `/api/video-tasks/{id}` | 查询任务详情 | 任务详情页 | 是 |
| API-005 | PUT | `/api/video-tasks/{id}` | 更新任务 | 任务详情页 | 是 |

## 4. 登录接口

### POST `/api/auth/login`

请求：

```json
{
  "username": "admin",
  "password": "password"
}
```

响应：

```json
{
  "code": "OK",
  "message": "success",
  "data": {
    "user": {
      "id": "string",
      "username": "string",
      "displayName": "string"
    }
  }
}
```

说明：登录成功后，由服务端设置登录 Cookie，响应体不返回 token。

## 5. 视频任务字段

| 字段 | 类型 | 是否必填 | 说明 |
|---|---|---|---|
| id | string | 是 | 任务 ID |
| title | string | 是 | 任务标题 |
| topic | string | 是 | 视频主题 |
| scriptSummary | string | 否 | 脚本摘要 |
| storyboardSummary | string | 否 | 分镜摘要 |
| status | string | 是 | draft / pending / generating / completed |
| createdAt | string | 是 | 创建时间 |
| updatedAt | string | 是 | 更新时间 |

## 6. 创建任务接口

### POST `/api/video-tasks`

请求：

```json
{
  "title": "2026 俄罗斯留学申请时间线",
  "topic": "俄罗斯留学申请",
  "scriptSummary": "介绍申请节奏和关键时间点",
  "storyboardSummary": "按时间轴展示"
}
```

响应：返回视频任务对象。

## 7. 更新任务接口

### PUT `/api/video-tasks/{id}`

请求：

```json
{
  "title": "string",
  "topic": "string",
  "scriptSummary": "string",
  "storyboardSummary": "string",
  "status": "draft"
}
```

响应：返回更新后的视频任务对象。

## 8. 错误码

| 错误码 | HTTP 状态 | 说明 | 前端行为 |
|---|---:|---|---|
| INVALID_REQUEST | 400 | 请求参数错误 | 展示字段或表单错误 |
| UNAUTHORIZED | 401 | 未登录或登录失效 | 提示登录失效并跳转登录页 |
| NOT_FOUND | 404 | 资源不存在 | 展示资源不存在提示 |
| INTERNAL_ERROR | 500 | 服务内部错误 | 展示系统错误提示 |
