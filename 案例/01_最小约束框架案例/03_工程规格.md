# 工程规格

## 1. 工程范围

第一版只做静态页面或轻量前端。

不做后端。

数据使用 Mock 数据或本地存储。

## 2. 数据字段

| 字段 | 类型 | 说明 |
|---|---|---|
| id | string | 地点唯一标识 |
| name | string | 地点名称 |
| description | string | 地点说明 |
| tags | string[] | 地点标签 |
| isFavorite | boolean | 是否已保存 |

## 3. 前端范围

- 展示地点列表
- 展示地点详情
- 切换保存状态
- 展示已保存列表
- 展示空状态

## 4. 后端范围

第一版不做后端。

## 5. 接口范围

第一版不做真实 API，使用 Mock 数据模拟地点列表。
