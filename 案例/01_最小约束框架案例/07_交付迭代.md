# 交付迭代

## 1. 交付条件

- 产品定义已确认
- 界面规格已确认
- 工程规格已确认
- 验证已通过
- 使用验收已通过

## 2. 部署检查

- 页面可打开
- 核心流程可完成
- 空状态可显示
- 没有引入不做清单里的能力

## 3. 反馈分类

| 类型 | 处理方式 |
|---|---|
| 缺陷 | 进入修正任务 |
| 使用问题 | 更新验收或界面规格 |
| 范围变化 | 先更新产品定义 |
| 新需求 | 放入下一版本评估 |
